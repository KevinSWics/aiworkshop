# AI Workshop - OpenEdge ABL Project

This project demonstrates the Business Entity architecture pattern for OpenEdge ABL applications.

## Structure

- `src/` - ABL source files (windows and business classes)
- `src/business/` - Business entity classes and dataset definitions
- `dump/` - Database schema
- `doc/` - Documentation
- `.windsurf/` - Windsurf AI assistant rules and workflows

## Getting Started

1. Open the project in Progress Developer Studio or VS Code with the ABL extension
2. Connect to the sports2000 database using `build.xml`
3. Run `src/CustomerWin.w` or `src/ItemWin.w` to see the UI
